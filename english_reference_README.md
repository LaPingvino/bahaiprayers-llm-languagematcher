# English Bahá'í Prayer Reference Collection

**Purpose**: Normative reference for matching prayers across languages using Phelps codes

**Source**: bahaiprayers.net and related sources  
**Total prayers**: 256  
**Last updated**: 2025-11-27  

**Phelps code format**: Two letters (author initials) + 5 digits + 3 letter mnemonic  
Example: `AB00001FIR` = 'Abdu'l-Bahá, prayer #1, "Fire tablet"

**Usage**: This file serves as the master reference for cross-language prayer matching. Each prayer in other languages should be assigned the same Phelps code to enable cross-linking without requiring translation.

**File format**: JSON with fields:
- `phelps`: Phelps inventory code
- `name`: Prayer name/title (if available)
- `text`: Full prayer text in English
- `category`: Prayer category/type

**Categories include**: prayer, healing, meditation, protection, etc.
